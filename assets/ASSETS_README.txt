REQUIRED ASSETS
===============

Please add the following image files to this folder:

1. icon.png (1024x1024) - App icon
2. splash.png (1284x2778) - Splash screen
3. adaptive-icon.png (1024x1024) - Android adaptive icon

You can create these using:
- Canva (free)
- Figma (free)
- Any image editor

Recommended Design (OfflineGPT Brand):
- Use the Gemini-style sparkle/star logo
- Gradient: Blue (#4b90ff) → Purple (#8ba4f9) → Red (#ff5546)
- Background: Black (#000000) for dark theme consistency
- OR use brand teal color: #10a37f

Logo SVG Path (for reference):
The logo is a Gemini-style sparkle pattern. You can use the SVG from
src/components/common/GeminiLogo.js to export as PNG.

Quick Option:
1. Open https://svg-to-png.com/
2. Copy the SVG from GeminiLogo.js
3. Export at 1024x1024 for icons
4. Export at 1284x2778 for splash (centered on black background)
